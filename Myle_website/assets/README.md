# Assets

Replace the SVG placeholder files in this folder with project screenshots, video stills, GIFs, or exported process images.

Keep file names simple for GitHub Pages, for example:

- `look-me-up-cover.jpg`
- `sound-galaxy-demo.gif`
- `grow-and-harvest-process.png`
- `call-your-mom-video-still.jpg`

Project page copy and image paths are managed in `js/main.js`.
